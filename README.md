# Actuarial Python Lab: Bridging the Gap Between Risk Theory and Software Engineering

## 🚀 Overview

**Actuarial Python Lab** is a comprehensive repository dedicated to the digital transformation of traditional actuarial science. While actuarial education provides a rigorous foundation in mathematics and statistics, the modern industry demands the ability to implement these models within scalable, automated, and reproducible software environments.

This repository serves as a bridge, translating complex actuarial theorems into production-ready Python code. Each module follows a strict **Theory → Application → Implementation** framework to demonstrate both subject matter expertise and technical proficiency.

---

## 🏗 Repository Structure

The project is organized into 10 core actuarial domains, covering both traditional insurance and "wider fields" like Banking, Health-Tech, and Data Science.

* **01 Life Contingencies:** Mortality modeling, premium pricing, and survival analysis.
* **02 General Insurance:** Loss reserving (IBNR), triangulation, and frequency-severity modeling.
* **03 Finance Mathematics:** Bond pricing, yield curve construction, and derivative Greeks.
* **04 Pensions:** Valuation of defined benefit/contribution schemes and funding projections.
* **05 Health Insurance:** Medical burning cost analysis and risk-sharing capitation models.
* **06 Banking & Credit:** Credit scoring, Probability of Default (PD), and Markov migrations.
* **07 Investment & ALM:** Portfolio optimization (Markowitz) and asset-liability matching.
* **08 Risk Theory:** Monte Carlo ruin simulations and Value at Risk (VaR).
* **09 Reinsurance:** Pricing for XoL (Excess of Loss) and Quota Share treaty optimization.
* **10 Actuarial Data Science:** GLMs, machine learning interpretability (SHAP), and IFRS 17 logic.

---

## 📖 Module Format

Every subdomain is documented via **Jupyter Notebooks** structured as follows:

1. **Actuarial Theory:** A deep dive into the underlying mathematical proofs and formulas using LaTeX (e.g., $A_x = \sum_{k=0}^{\infty} v^{k+1} {}_k p_x q_{x+k}$).
2. **Real-World Application:** A business-case analysis explaining how the theory solves problems in the Kenyan and global financial markets.
3. **Python Implementation:** Optimized code using the scientific Python stack (`NumPy`, `Pandas`, `SciPy`) and visualization libraries (`Matplotlib`, `Seaborn`).
4. **Verification & Testing:** Unit tests to ensure the programmatic output aligns with recognized actuarial tables and benchmarks.
